# Tic Tac Toe (Крестики-нолики онлайн)

### Tools (Инструменты)
- python >= 3.10
- Starlette
- WebSockets
- Uvicorn

### Start (Старт)

    pip install -r requirements.txt
    uvicorn main:app --reload

### Start with Docker (Старт через Docker)

    docker-compose build
    docker-compose up

    or

    docker-compose up --build
